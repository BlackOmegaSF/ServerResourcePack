# Server Resource Pack

A custom resource pack for my personal Minecraft server.

## Sounds

Music disc `cat` now rickrolls you.

Custom sounds:
- `kleinercode/whip`
- `kleinercode/oof`

## Model and Texture Changes
- Shields take up less space on the screen when wielded.
- Iron Bars texture is replaced with the one from JimStoneCraft 2020
- Paintings `burning_skull` and `void` are replaced with Middle Finger Mario.
